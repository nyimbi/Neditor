# Client Context Include {#sec:client-context}

{{client}} wants a document system that can support board decisions, RFP
responses, research reports, textbooks, tutorials, policy briefs, novels,
podcast scripts, movie scripts, publishing handoffs, and release-ready business
documents without turning every writer into a developer.

```glossary
Local-first: A product design where the user's files remain owned, inspectable, and portable on the local machine or chosen workspace.
Export manifest: A sidecar evidence file that records source hashes, include hashes, options, diagnostics, and generated artifact details.
Compliance matrix: A table that maps every buyer requirement to the response section, evidence owner, and verification method.
AI provenance: Metadata that records which AI system helped draft content and whether humans reviewed it.
```
